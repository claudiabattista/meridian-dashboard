R1 — Reports Remediation

Fix it completely.
Not just enough to pass.

The previous vendor's handoff acknowledged the Reports module was incomplete. "At least eight issues" is a floor, not a ceiling. Our approach:

  • Full defect audit in Week 1 — produce a shared list before writing code
  • Wire all filters end-to-end: Vue → API client → FastAPI → filtered response
  • Verify every visible string is in the translation files — no English fallback in production
  • Acceptance criterion: correct results on every filter, zero console errors
